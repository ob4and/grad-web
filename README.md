**Digital Graduation Exhibition Platform (2024)**

## Overview
This project is the official exhibition website for the  
**17th Kongju University Design Convergence Graduation Exhibition**,  
featuring 44 students and over 110 works across two majors.

The website was designed as a digital exhibition platform focused on  
clear navigation, equal visibility of student works, and scalability  
for large-scale exhibition content.

## Key Design Focus
- **System-first Exhibition Design**  
  Defined the exhibition website as an operational system rather than a static outcome.
- **Scalable Information Architecture**  
  Designed an archive-centered structure that supports expansion by project and designer.
- **Equal Representation**  
  Ensured all student works were displayed with consistent hierarchy and visibility.

## Outcome
- Fully responsive exhibition website  
- Archive-based project navigation  
- Designer profile pages  
- Live operation during the exhibition period
